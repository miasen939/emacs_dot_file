实现home row mods
目前的配置
- a; super
- sl meta
- dk ctrl 
- fj shift
- gh C-M-
- capslock :: hold as ctrl, tap as esc
- lctrl :: capslock
使用fj 当做shift的话，最好是保留键盘上的大写锁，这样输入如 HTTP 时，就会方便很多
